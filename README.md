# bach_visualization

This is a facsimile of [chen alexander](http://www.chenalexander.com/).

Dedicated to Bach. A visual art for Bach's Suites for Solo Cello, Suite No.1, BWV 1007, in G/G-dur/sol majeur. Prelude.
